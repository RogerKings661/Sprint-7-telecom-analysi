# Sprint-7-telecom-analysis
## Objetivos del proyecto
Ayudar a ConnectaTel a entender cómo se comportan sus usuarios según tres dimensiones clave:

-Edad de los usuarios

-Volumen de llamadas y mensajes que utilizan

-Nivel de consumo general
## Datasets Utilizados
Se trabajo con 3 datasets diferentes

-plans.csv: los planes actuales (precio, minutos incluidos, GB incluidos, costo por extra).

-users_latam.csv: información de clientes: edad, ciudad, fecha de registro, plan contratado.

-usage.csv: el detalle de uso real: llamadas (duración) y mensajes (longitud).
## Etapas de analisis realizadas
A través de un análisis completo de datos que incluyó:

-Limpieza de datos (eliminar inconsistencias)

-Análisis estadístico (medias, medianas, distribuciones)

-Detección de outliers (usuarios con comportamientos extremos)

-Segmentación de usuarios (agrupar por patrones de uso)
